- **매수/업그레이드**: 자신의 땅에 착지했을 때(`phase === 'on_my_prop'`)만 가능

### 통행료 계산

```
통행료 = CELLS[cellId].rent[entry.buildings]
```

상대방 소유 건물 레벨(0~3)에 따라 결정. 황금열쇠 아이템 보유 시 1회 면제.

> **주의**: `rent[]` 배열은 6개 원소이나, 실제 건물 레벨 최대치(3)까지만 사용됨.

## 대출 시스템

- **한도**: 보유 부동산 매도가 합계 × **LTV 70%**
- **금리**: 연 4.5% → 바퀴당 이자 = 잔액 × (4.5% ÷ 12)
- **이자 납부 시점**: 출발점 통과(바퀴 완주) 시 자동 납부
- **대출 선택**: 가능액의 25% / 50% / 75% / 100%
- **상환 선택**: 잔액의 25% / 50% / 100%
- 이자 납부 후 현금이 마이너스면 파산 판정

## 순자산 계산식

```
순자산 = max(0, 현금) + Σ(보유 부동산 매도가) − 대출잔액
```

> 현금이 마이너스일 때 파산이지만, 순자산 계산 시 현금은 0으로 처리.

## AI 행동 패턴

1. **건물 업그레이드**: 보유 자금이 업그레이드 비용의 2배 이상일 때 자동 매입
2. **대출**: 원하는 부동산이 있지만 자금이 부족하고 대출로 가능할 때 자동 실행
3. **부동산 매수**: 착지 시 자금이 매매가의 1.2배 이상이면 최고 레벨로 즉시 매수
4. **청약**: 가장 싼 미소유 부동산부터 빌라 반값 매수
